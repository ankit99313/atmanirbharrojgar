# SHRAM - An  Android  app with Material Design Pattern Using Firebase ,google map API, messaging API
## Shram is basically an android application where daily wage workers can get the opportunity to get the daily job.


Firebase realtime database has been used for user registration and login whereas firebase cloud storage is being user for storing profile images of the user etc !

## AIM of the SHARM PROJECT

The aim of this project is to develop Android based application where a low level worker
can get a work and a normal person can get skilled worker, to get his work done. The
platform should benefit society by improving communication between low level worker
and normal person and able to provide a proper structure for giving and taking work at
lower level.
## The scope of this project SHRAM includes:
- The SHRAM provides the opportunity of job to the daily wage workers on
   day to day basis.
- Provides impartial platform for giving and taking work at lower level.
- Supported by well-designed database and Android based design.
- A friendly User Interface is provided to facilitate different user services


## Features Integrated
- List Of Available Service
- Profiles Page
- Filter Based on Location
